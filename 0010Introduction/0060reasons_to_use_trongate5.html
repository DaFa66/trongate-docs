Reasons to Use Trongate

Reason #5: It's Easy!

As stated on the previous page, Trongate favors pure PHP over third-party dependencies. This results in a syntax and architecture that are simple and robust.

However, we've gone even further by producing a best-in-class code generator called 'The Trongate Desktop App'.

With the Trongate Desktop App, developers can easily build large-scale, database-driven applications in a remarkably short period of time. In addition to providing developers with a beautiful code generator, we've also added a graphical query builder to help developers simplify complex table joins.

All of this is available for free and is fully open source.
