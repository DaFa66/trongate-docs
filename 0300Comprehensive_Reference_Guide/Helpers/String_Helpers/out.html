out()

function out(?string $input, string $encoding = 'UTF-8', string $output_format = 'html'): string

Description

Safely escape and format a string for various output contexts. This function supports multiple output formats including HTML, XML, JSON, JavaScript, and HTML attributes. It throws an exception if an unsupported output format is provided.

Parameters

Parameter Type Description Default
$input string|null The string to be escaped. null
$encoding string The character encoding to use for escaping. 'UTF-8'
$output_format string Specifies the output format ('html', 'xml', 'json', 'javascript', 'attribute'). 'html'

Return Value

Type Description
string Returns the escaped and formatted string for safe inclusion in the specified context, or throws an exception for unsupported formats.

Example Usage

echo out("<Hello World!>", "UTF-8", "html");
// Output: &lt;Hello World!&gt;
