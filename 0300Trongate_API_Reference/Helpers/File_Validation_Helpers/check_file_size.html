function check_file_size(int $file_size, int $file_check_value): string

Description

Checks if the file size exceeds a specified limit.

Parameters

Parameter Type Description Default
$file_size int The size of the file to be checked, in kilobytes. N/A
$file_check_value int The maximum allowed file size, in kilobytes. N/A

Return Value

Type Description
string A message indicating if the file size exceeds the limit. If not, an empty string is returned.

Example Usage

$file_size = 1024; // 1MB
$file_check_value = 512; // 512KB (0.5MB)
$result = check_file_size($file_size, $file_check_value);
if (!empty($result)) {
    echo $result;
} else {
    echo "File size is within the allowed limit.";
}
