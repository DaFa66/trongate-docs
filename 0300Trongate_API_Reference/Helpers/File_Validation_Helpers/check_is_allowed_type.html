function check_is_allowed_type(string $target_file_name, string $file_check_value): string

Description

Checks if the file type is allowed based on specified types.

Parameters

Parameter Type Description Default
$target_file_name string The name of the target file to check. N/A
$file_check_value string A comma-separated string containing allowed file types. N/A

Return Value

Type Description
string A message indicating if the file type is not allowed. If the file type is allowed, an empty string is returned.

Example Usage

$target_file_name = "example.jpg";
$file_check_value = "jpg,png,gif";
$result = check_is_allowed_type($target_file_name, $file_check_value);
if (!empty($result)) {
    echo $result;
} else {
    echo "File type is allowed.";
}
