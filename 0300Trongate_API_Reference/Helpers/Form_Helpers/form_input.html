form_input()

function form_input(string $name, mixed $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element.

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value mixed (optional) The initial value for the input element. Default is null.
$attributes array|null (optional) Additional attributes for the input element. Default is null.
$additional_code string|null (optional) Additional code to include in the input element. Default is null.

Return Value

Type Description
string The HTML representation of the input element.

Example Usage

$name = 'username';
$value = 'john_doe';
$attributes = ['class' => 'form-control', 'id' => 'username-input'];
$additional_code = 'placeholder="Enter your username"';
echo form_input($name, $value, $attributes, $additional_code);
// Output: '<input type="text" name="username" value="john_doe" class="form-control" id="username-input" placeholder="Enter your username">'
