form_textarea()

function form_textarea(string $name, string|null $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML textarea element.

Parameters

Parameter Type Description
$name string The name attribute for the textarea element.
$value string|null The initial value of the textarea. If not provided, it will be empty.
$attributes array|null An associative array of HTML attributes for the textarea.
$additional_code string|null Additional HTML code to append to the textarea element.

Return Value

Type Description
string The generated HTML textarea element.

Example Usage

  $name = 'message';
  $value = 'Enter your message here';
  $attributes = ['rows' => 4, 'cols' => 50];
  $additional_code = 'class="form-control"';
  echo form_textarea($name, $value, $attributes, $additional_code);
  // Output: '<textarea name="message" rows="4" cols="50" class="form-control">Enter your message here</textarea>'
