form_number()

function form_number(string $name, int|float|string|null $value = null, ?array $attributes = null, ?string $additional_code = null): ?string

Description

Generates an HTML input element with type "number".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value int|float|string|null The initial value of the input element.
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string|null The generated HTML input element with type "number", or null if not generated.

Example Usage

$name = 'quantity';
$value = 10;
$attributes = ['min' => 1, 'max' => 100];
$additional_code = 'step="1"';
echo form_number($name, $value, $attributes, $additional_code);
// Output: '<input type="number" name="quantity" value="10" min="1" max="100" step="1">'
