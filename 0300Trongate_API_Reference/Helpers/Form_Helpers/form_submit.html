form_submit()

function form_submit(string $name, string|null $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML submit button element.

Parameters

Parameter Type Description
$name string The name attribute for the button element.
$value string|null The value of the button. If not provided, the button's name will be used as the value.
$attributes array|null An associative array of HTML attributes for the button.
$additional_code string|null Additional HTML code to append to the button element.

Return Value

Type Description
string The generated HTML submit button element.

Example Usage

  $name = 'submit_button';
  $value = 'Submit';
  $attributes = ['class' => 'btn btn-primary'];
  $additional_code = 'id="submitBtn"';
  echo form_submit($name, $value, $attributes, $additional_code);
  // Output: '<button type="submit" name="submit_button" value="Submit" class="btn btn-primary" id="submitBtn">Submit</button>'
