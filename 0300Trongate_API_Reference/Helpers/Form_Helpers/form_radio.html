form_radio()

function form_radio(string $name, string|null $value = null, bool $checked = false, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element with type "radio".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value string|null The value of the radio button.
$checked bool Whether the radio button should be checked (true) or unchecked (false).
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string The generated HTML input element with type "radio".

Example Usage

$name = 'gender';
$value = 'male';
$checked = true;
$attributes = ['class' => 'radio-btn'];
$additional_code = 'id="maleRadio"';
echo form_radio($name, $value, $checked, $attributes, $additional_code);
// Output: '<input type="radio" name="gender" value="male" checked class="radio-btn" id="maleRadio">'
