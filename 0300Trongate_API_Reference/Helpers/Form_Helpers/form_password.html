form_password()

function form_password(string $name, string|null $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element with type "password".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value string|null The initial value of the input element (password). Use null for no initial value.
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string The generated HTML input element with type "password".

Example Usage

$name = 'password';
$value = '';
$attributes = ['placeholder' => 'Enter your password'];
$additional_code = 'required';
echo form_password($name, $value, $attributes, $additional_code);
// Output: '<input type="password" name="password" value="" placeholder="Enter your password" required>'
