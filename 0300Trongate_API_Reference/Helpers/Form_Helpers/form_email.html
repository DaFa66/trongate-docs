form_email()

function form_email(string $name, string|null $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element with type "email".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value string|null The initial value of the input element (email). Use null for no initial value.
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string The generated HTML input element with type "email".

Example Usage

  $name = 'email';
  $value = '';
  $attributes = ['placeholder' => 'Enter your email'];
  $additional_code = 'required';
  echo form_email($name, $value, $attributes, $additional_code);
  // Output: '<input type="email" name="email" value="" placeholder="Enter your email" required>'
