form_search()

function form_search(string $name, $value = null, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML search input element. This function creates a search input field allowing users to input search queries.

Parameters

Parameter Type Description Default Required
$name string The name attribute for the search input element. - Yes
$value mixed The initial value for the search input element. Default is null. null No
$attributes array|null Additional attributes for the search input element. Default is null. null No
$additional_code string|null Additional code to include in the search input element. Default is null. null No

Return Value

Type Description
string The HTML representation of the search input element.

Example Usage

$search_attr['placeholder'] = 'Enter search phrase here...';
$search_attr['autocomplete'] = 'off';
echo form_search('searchphrase', '', $search_attr);
