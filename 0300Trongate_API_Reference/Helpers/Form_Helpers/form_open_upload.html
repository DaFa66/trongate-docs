form_open_upload()

function form_open_upload(string $location, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates the opening tag for an HTML form with file upload support.

Parameters

Parameter Type Description Default
$location string The URL to which the form will be submitted. N/A
$attributes array|null An optional array of HTML attributes for the form. null
$additional_code string|null An optional additional code to include in the form tag. null

Return Value

Type Description
string The HTML opening tag for the form with enctype set to "multipart/form-data."

Example Usage

$location = "/submit-form";
$attributes = ['class' => 'my-form', 'id' => 'form-id'];
$additional_code = 'data-ajax="true"';
$form_opening_tag = form_open_upload($location, $attributes, $additional_code);
echo $form_opening_tag;
// Output: <form action="/submit-form" class="my-form" id="form-id" data-ajax="true" enctype="multipart/form-data">
