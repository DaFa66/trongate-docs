form_checkbox()

function form_checkbox(string $name, string|null $value = null, bool $checked = false, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML input element with type "checkbox".

Parameters

Parameter Type Description
$name string The name attribute for the input element.
$value string|null The value of the checkbox when checked.
$checked bool Whether the checkbox should be checked (true) or unchecked (false).
$attributes array|null An associative array of HTML attributes for the input.
$additional_code string|null Additional HTML code to append to the input element.

Return Value

Type Description
string The generated HTML input element with type "checkbox".

Example Usage

  $name = 'subscribe';
  $value = 'yes';
  $checked = true;
  $attributes = ['class' => 'checkbox'];
  $additional_code = 'id="subscribeCheckbox"';
  echo form_checkbox($name, $value, $checked, $attributes, $additional_code);
  // Output: '<input type="checkbox" name="subscribe" value="yes" checked class="checkbox" id="subscribeCheckbox">'
