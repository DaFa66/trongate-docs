form_file_select()

function form_file_select(string $name, array|null $attributes = null, string|null $additional_code = null): string

Description

Generates an HTML file input element.

Parameters

Parameter Type Description
$name string The name attribute for the file input.
$attributes array|null An array of HTML attributes for the file input.
$additional_code string|null Additional HTML code to include in the file input.

Return Value

Type Description
string The generated HTML for the file input element.

Example Usage

$name = 'file';
$attributes = ['class' => 'form-control'];
$additional_code = 'accept="image/png, image/jpeg"';
echo form_file_select($name, $attributes, $additional_code);
// Output: '<input type="file" name="file" class="form-control" accept="image/png, image/jpeg">'
