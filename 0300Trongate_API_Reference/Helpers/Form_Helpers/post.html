post()

function post(string $field_name, bool|null $clean_up = null): string|int|float

Description

Retrieves and cleans a value from the POST data.

Parameters

Parameter Type Description
$field_name string The name of the POST field to retrieve.
$clean_up bool|null Optional. Whether to clean up the retrieved value. Default is null.

Return Value

Type Description
string|int|float The value retrieved from the POST data.

Example Usage

$field_name = 'username';
$clean_up = true;
echo post($field_name, $clean_up);
// Output: Cleaned up value of the 'username' field from the POST data.
