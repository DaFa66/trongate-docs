form_label()

function form_label(string $label_text, ?array $attributes = null, ?string $additional_code = null): string

Description

Generates an HTML label element with optional attributes.

Parameters

Parameter Type Description
$label_text string The text to display inside the label.
$attributes array|null An associative array of HTML attributes for the label.
$additional_code string|null Additional HTML code to append to the label element.

Return Value

Type Description
string The generated HTML label element.

Example Usage

  $label_text = 'Email Address:';
  $attributes = ['for' => 'email-input', 'class' => 'form-label'];
  $additional_code = 'data-required="true"';
  echo form_label($label_text, $attributes, $additional_code);
  // Output: '<label for="email-input" class="form-label" data-required="true">Email Address:</label>'
