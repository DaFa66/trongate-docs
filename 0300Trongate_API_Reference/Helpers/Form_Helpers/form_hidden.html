form_hidden()

function form_hidden(string $name, string|null $value = null, ?string $additional_code = null): string

Description

Generates an HTML hidden input field.

Parameters

Parameter Type Description
$name string The name attribute for the hidden input field.
$value string|null The initial value of the hidden input field. If not provided, it will be empty.
$additional_code string|null Additional HTML code to append to the hidden input field.

Return Value

Type Description
string The generated HTML hidden input field.

Example Usage

$name = 'user_id';
$value = '123';
$additional_code = 'class="hidden-field"';
echo form_hidden($name, $value, $additional_code);
// Output: '<input type="hidden" name="user_id" value="123" class="hidden-field">'
