get_attributes_str()

function get_attributes_str(?array $attributes): string

Description

Gets a string representation of HTML attributes from an associative array.

Parameters

Parameter Type Description
$attributes array|null An associative array of HTML attributes.

Return Value

Type Description
string A string representation of HTML attributes.

Example Usage

  $attributes = [
      'class' => 'btn btn-primary',
      'id' => 'submit-btn',
      'disabled' => true
  ];
  $attributes_str = get_attributes_str($attributes);
  echo $attributes_str;
  // Output: ' class="btn btn-primary" id="submit-btn" disabled="disabled"'
