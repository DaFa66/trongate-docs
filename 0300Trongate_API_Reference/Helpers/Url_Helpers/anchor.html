anchor()

function anchor(string $target_url, mixed $text, array|null $attributes = null, string|null $additional_code = null): string

Description

Generates an HTML anchor (link) element.

Parameters

Parameter Type Description Default
$target_url string The URL to link to. N/A
$text mixed The link text or boolean value to indicate no link. N/A
$attributes array|null Optional. An associative array of HTML attributes for the anchor element. null
$additional_code string|null Optional. Additional HTML code to append to the anchor element. null

Return Value

Type Description
string Returns the HTML anchor element as a string.

Example Usage

  echo anchor("https://example.com", "Click me", ['class' => 'btn', 'target' => '_blank'], 'data-confirm="Are you sure?"');
  // Output: <a href="https://example.com" class="btn" target="_blank" data-confirm="Are you sure?">Click me</a>
