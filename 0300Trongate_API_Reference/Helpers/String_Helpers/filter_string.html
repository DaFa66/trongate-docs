filter_string()

function filter_string(string $string, string[] $allowed_tags = []): string

Description

Filters and sanitizes a string.

Parameters

Parameter Type Description
$string string The input string to be filtered and sanitized.
$allowed_tags string[] Optional. An array of allowed HTML tags. Default is an empty array.

Return Value

Type Description
string The filtered and sanitized string.

Example Usage

  $input_string = '<script>alert("Hello");</script>';
  $allowed_tags = ['<p>', '<a>'];
  echo filter_string($input_string, $allowed_tags);
  // Output: 'alert("Hello");'
