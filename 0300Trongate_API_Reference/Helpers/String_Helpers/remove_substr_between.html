function remove_substr_between(string $start, string $end, string $haystack): string

Description

Removes a portion of a string between two given substrings.

Parameters

Parameter Type Description Default Required
$start string The starting substring. N/A Yes
$end string The ending substring. N/A Yes
$haystack string The string from which to remove the substring. N/A Yes

Return Value

Type Description
string The modified string.

Example Usage

$start = 'h';
$end = 'o';
$haystack = 'hello';
echo remove_substr_between($start, $end, $haystack);
// Output: 'ell'
