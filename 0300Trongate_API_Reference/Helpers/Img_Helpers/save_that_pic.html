save_that_pic()

private function save_that_pic(array $data): void

Description

Resizes and saves an image file based on provided parameters such as compression, permissions, maximum width, and maximum height.

Parameters

Parameter Type Description
$data array An associative array containing information necessary for resizing and saving the image.

Return Value

This method does not return any value; it performs operations on the image file directly.

Example Usage

  $data = [
      'new_file_path' => 'uploads/image.jpg',
      'compression' => 80,
      'permissions' => 755,
      'max_width' => 800,
      'max_height' => 600,
      'tmp_file_width' => 1024,
      'tmp_file_height' => 768,
      'image' => new Image('uploads/tmp_image.jpg'),
  ];
  save_that_pic($data);
