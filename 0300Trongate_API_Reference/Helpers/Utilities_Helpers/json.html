json()

function json(mixed $data, bool|null $kill_script = null): void

Description

Encodes data as JSON and optionally displays it with preformatted HTML.

Parameters

Parameter Type Description Default
$data mixed The data to be encoded as JSON. N/A
$kill_script bool|null Optional. If true, terminate the script after displaying the JSON. Default is null. null

Return Value

This function does not return a value; it echoes the JSON data.

Example Usage

json(['name' => 'John', 'age' => 30]);
// Output: {
//   "name": "John",
//   "age": 30
// }
