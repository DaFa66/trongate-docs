handle_empty_file_error()

private function handle_empty_file_error(string $key, string $label): void

Description

Handles the error for an empty file.

Parameters

Parameter Type Description
$key string The key associated with the file input.
$label string The label for the file input.

Return Value

This method does not return a value.

Example Usage

handle_empty_file_error("profile_picture", "Profile Picture");
