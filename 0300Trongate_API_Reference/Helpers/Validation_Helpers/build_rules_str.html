build_rules_str()

private function build_rules_str(array $value): string

Description

Build rules string based on the provided validation rules.

Parameters

Parameter Type Description
$value array An array representing validation rules.

Return Value

Returns a string containing rules generated from the validation rules.

Example Usage

$validation_rules = [
    'required' => true,
    'min_length' => 5,
    // other validation rules
];
$rules_str = $validation->build_rules_str($validation_rules);
echo $rules_str; // Output: required|min_length[5]
