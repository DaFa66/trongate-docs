valid_datepicker_us()

private function valid_datepicker_us(array $validation_data): bool

Description

Validates a datepicker value by invoking the valid_datepicker() function.

This function serves as an alias for valid_datepicker().

Parameters

Parameter Type Description
$validation_data array An array containing validation data.

Return Value

Returns true if the datepicker value is valid, false otherwise.

Throws

Exception - If the posted value is not a valid date in the expected format.

Example Usage

$validation_data = [
    'posted_value' => $_POST['datepicker_field'],
    'key' => 'datepicker_field',
    'label' => 'Datepicker Field'
];
$isValid = $validation->valid_datepicker_us($validation_data);
if (!$isValid) {
    // handle error
}
