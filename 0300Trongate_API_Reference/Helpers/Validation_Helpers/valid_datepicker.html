valid_datepicker()

private function valid_datepicker(array $validation_data): bool

Description

Validates a datepicker value.

Required keys in the validation data array: 'posted_value', 'key', 'label'

Parameters

Parameter Type Description
$validation_data array An array containing validation data.

Return Value

Returns true if the datepicker value is valid, false otherwise.

Throws

Exception - If the posted value is not a valid date in the expected format.

Example Usage

$validation_data = [
    'posted_value' => $_POST['datepicker_field'],
    'key' => 'datepicker_field',
    'label' => 'Datepicker Field'
];
$isValid = $validation->valid_datepicker($validation_data);
if (!$isValid) {
    // handle error
}
