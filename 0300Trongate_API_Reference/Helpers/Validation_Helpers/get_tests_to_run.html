get_tests_to_run()

private function get_tests_to_run(string $rules): array

Description

Get tests to run based on provided rules.

Parameters

Parameter Type Description Default
$rules string A string containing validation rules separated by '|'. N/A

Return Value

An array containing individual tests to run based on rules.

Example Usage

$validation_rules = 'required|min_length[5]|max_length[20]';
$tests_to_run = $validation->get_tests_to_run($validation_rules);
print_r($tests_to_run); // Output: ['required', 'min_length[5]', 'max_length[20]']
