run_file_validation()

private function run_file_validation(string $key, mixed $rules): void

Description

Runs the file validation logic.

Parameters

Parameter Type Description
$key string The key associated with the file input.
$rules mixed The rules for file validation.

Return Value

This method does not return a value.

Example Usage

run_file_validation("profile_picture", ["required", "max_size[1024]"]);
