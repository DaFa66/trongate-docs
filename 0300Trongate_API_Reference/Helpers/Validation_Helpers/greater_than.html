greater_than()

private function greater_than(string $key, string $label, string $posted_value, int $inner_value): void

Description

Validates if the posted value is greater than a specified inner value.

Parameters

Parameter Type Description
$key string The key associated with the posted value.
$label string The label/name of the posted value.
$posted_value string The posted value to be compared.
$inner_value int The value for comparison.

Return Value

This method doesn't return a value; it adds an error message to the form submission errors if the posted value is not greater than the specified inner value.

Example Usage

$validation->greater_than('age', 'Age', $_POST['age'], 18);
