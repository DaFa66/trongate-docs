valid_datetimepicker_eu()

private function valid_datetimepicker_eu(array $validation_data): bool

Description

Alias of valid_datetimepicker().

Parameters

Parameter Type Description
$validation_data array The validation data containing key, label, and posted value.

Return Value

Returns true if the input is a valid date and time, otherwise adds an error message and returns false.

Example Usage

$validation_data = [
    'posted_value' => $_POST['datetimepicker_field'],
    'key' => 'datetimepicker_field',
    'label' => 'Datetimepicker Field'
];
$isValid = $validation->valid_datetimepicker_eu($validation_data);
if (!$isValid) {
    // handle error
}
