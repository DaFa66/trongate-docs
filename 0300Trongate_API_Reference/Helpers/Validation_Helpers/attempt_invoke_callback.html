attempt_invoke_callback()

private function attempt_invoke_callback(string $key, string $label, mixed $posted_value, string $test_to_run): void

Description

Attempts to invoke a callback method for validation.

Parameters

Parameter Type Description
$key string The key associated with the input field.
$label string The label for the input field.
$posted_value mixed The value posted for validation.
$test_to_run string The name of the test to run.

Return Value

This method does not return a value.

Example Usage

attempt_invoke_callback("username", "Username", "test_username", "callback_username_available");
