valid_email()

private function valid_email(array $validation_data): void

Description

Validates the provided email address.

Parameters

Parameter Type Description
$validation_data array The validation data containing key, label, and posted value.

Return Value

This method doesn't return a value; it adds an error message to the form submission errors if the email address is not valid.

Example Usage

$validation->valid_email(['key' => 'email', 'label' => 'Email', 'posted_value' => $_POST['email']]);
