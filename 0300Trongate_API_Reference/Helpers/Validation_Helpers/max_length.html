max_length()

private function max_length(string $key, string $label, string $posted_value, int $inner_value): void

Description

Validates if the posted value meets a maximum length requirement.

Parameters

Parameter Type Description
$key string The key associated with the posted value.
$label string The label/name of the posted value.
$posted_value string The posted value to be checked.
$inner_value int The maximum length requirement.

Return Value

This method doesn't return a value; it adds an error message to the form submission errors if the posted value exceeds the maximum length requirement.

Example Usage

$validation->max_length('password', 'Password', $_POST['password'], 20);
