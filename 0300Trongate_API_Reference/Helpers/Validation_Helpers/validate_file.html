validate_file()

private function validate_file(string $key, string $label, mixed $rules): void

Description

Validates a file based on the provided rules.

Parameters

Parameter Type Description
$key string The key associated with the file input.
$label string The label for the file input.
$rules mixed The rules for file validation.

Return Value

This method does not return a value.

Example Usage

validate_file("profile_picture", "Profile Picture", ['allowed_types' => 'jpg|png', 'max_size' => 2048]);
