_get_test_name()

private function _get_test_name(string $test_to_run): string

Description

Gets the test name from the test to run string containing square brackets.

Parameters

Parameter Type Description
$test_to_run string The string containing the test name and parameters.

Return Value

Returns the extracted test name as a string.

Example Usage

$test_to_run = "matches[password_confirmation]";
echo _get_test_name($test_to_run);
// Output: matches
