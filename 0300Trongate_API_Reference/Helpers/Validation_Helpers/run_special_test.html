run_special_test()

private function run_special_test(array $validation_data): void

Description

Runs a special validation test based on the provided test name and value within square brackets.

Parameters

Parameter Type Description
$validation_data array The validation data containing key, label, posted value, and test to run.

Return Value

This method doesn't return a value; it invokes specific validation methods based on the provided test name and value.

Example Usage

$validation->run_special_test([
    'key' => 'email',
    'label' => 'Email',
    'posted_value' => $_POST['email'],
    'test_to_run' => 'valid_email'
]);
