run_validation_test()

private function run_validation_test(array $validation_data, mixed|null $rules = null): void

Description

Run a validation test based on the provided validation data and rules.

Parameters

Parameter Type Description Default
$validation_data array An array containing validation data. N/A
$rules mixed|null The rules for validation. null

Return Value

This method does not return any value (void).

Example Usage

$validation_data = [
    'test_to_run' => 'required',
    // other validation data
];
$rules = ['required', 'valid_email'];
$validation->run_validation_test($validation_data, $rules);
