check_for_integer()

private function check_for_integer(array $validation_data): void

Description

Check if the value in validation data is an integer.

Required keys in the validation data array: 'key', 'label', 'posted_value'

Parameters

Parameter Type Description
$validation_data array An array containing validation data.

Return Value

This method does not return any value (void).

Example Usage

$validation_data = [
    'key' => 'field_name',
    'label' => 'Field Label',
    'posted_value' => $_POST['field_name']
];
$validation->check_for_integer($validation_data);
