validation_errors()

function validation_errors(?string $opening_html = null, ?string $closing_html = null): ?string

Description

Retrieve and display validation error messages.

Parameters

Parameter Type Description Default
$opening_html string|null Optional. The opening HTML tag for displaying individual field errors. null
$closing_html string|null Optional. The closing HTML tag for displaying individual field errors. null

Return Value

Type Description
string|null Returns the formatted validation error messages or null if no errors exist.

Example Usage

// Example 1: Displaying individual field errors with custom opening and closing HTML tags
echo validation_errors('div', '
'); // Example 2: Normal error reporting with default opening and closing HTML tags echo validation_errors();
