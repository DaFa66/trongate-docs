differs()

private function differs(string $key, string $label, string $posted_value, string $target_field): void

Description

Validates if the posted value differs from a specified target field's value.

Parameters

Parameter Type Description
$key string The key associated with the posted value.
$label string The label/name of the posted value.
$posted_value string The posted value to be compared.
$target_field string The target field to compare against.

Return Value

This method doesn't return a value; it adds an error message to the form submission errors if the posted value matches the specified target field's value.

Example Usage

$validation->differs('password', 'Password', $_POST['password'], 'username');
