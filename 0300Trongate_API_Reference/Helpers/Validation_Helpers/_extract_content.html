_extract_content()

private function _extract_content(string $string, string $start, string $end): string

Description

Extracts content between specified start and end strings within a given string.

Parameters

Parameter Type Description
$string string The input string to search within.
$start string The starting string to search for.
$end string The ending string to search for.

Return Value

Returns the extracted content as a string.

Example Usage

$input_string = "This is a [test] string for extraction.";
$start = "[";
$end = "]";
echo _extract_content($input_string, $start, $end);
// Output: test
