exec()

public function exec(string $sql): void

Description

This method is used to execute SQL code. It's primarily intended for usage by Trongate's Module Import Wizard and should not be used in production environments. To execute custom SQL queries within normal operational conditions, the query() method is recommended.

Parameters

Parameter Type Description Default Required
sql string The SQL statement to execute. N/A Yes

Throws

Exception Description
Exception If the application environment is not set to 'dev'.
PDOException If an error occurs during the database operation.

Note

This method should only be used for development purposes and may produce undesired consequences if used improperly. It is disabled in production environments.
