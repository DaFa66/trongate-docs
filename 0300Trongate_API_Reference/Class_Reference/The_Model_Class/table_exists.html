table_exists()

public function table_exists(string $table_name): bool

Description

Checks if a table exists in the database. Returns true if the table exists and false if the table does not exist.

Parameters

Parameter Type Description Default Required
$table_name string The name of the table to check. N/A Yes

Return Value

Type Description
bool Returns true if the table exists, false otherwise.

Example Usage

$table_exists = $this->model->table_exists('products');
