run()

public function run(?array $validation_array = null): ?bool

Description

Run form validation checks.

Parameters

Parameter Type Description Default
$validation_array array|null An array containing validation rules. null

Return Value

Type Description
bool|null Returns a boolean value if validation completes, null if the script execution is potentially terminated.

Example Usage

$validation_array = [
    // validation rules
];

$result = $this->validation->run($validation_array);

if ($result === true) {
    // validation passed
} else {
    // validation failed
}
