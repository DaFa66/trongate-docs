download()

function download(string $file_path, bool $as_attachment = true): void

Description

Initiates a file download or displays inline from the server or an external URL.

This method prepares and sends headers based on the parameters to either initiate a file download from the server's local storage or display it inline. It checks if the file exists and is readable before proceeding.

The $as_attachment parameter determines whether the file should be downloaded as an attachment or displayed inline. If set to true, the browser will prompt the user to download the file. If set to false, the file will be displayed directly in the browser, if supported.

Parameters

Parameter Type Description Default
$file_path string The path or URL of the file. N/A
$as_attachment bool Determines whether to force the file download (true) or display inline (false). true

Throws

Exception if the file does not exist, cannot be read, or access to it is restricted.

Return Value

void

Example Usage

$file_path = '/path/to/your/file.txt';
try {
    // Force download
    $this->file->download($file_path);

    // Display inline
    $this->file->download($file_path, false);
} catch (Exception $e) {
    echo 'Error: ' . $e->getMessage();
}
