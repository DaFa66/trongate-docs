copy()

function copy(string $source_path, string $destination_path): bool

Description

Copies a file from one location to another.

Parameters

Parameter Type Description Default
$source_path string The path to the source file. N/A
$destination_path string The path to the destination where the file will be copied. N/A

Return Value

Type Description
bool Returns true on success, or false on failure.

Example Usage

copy("/path/to/source/file.txt", "/path/to/destination/file.txt");
// Output: true (success) or throws exception if fails
