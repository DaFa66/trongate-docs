info()

function info(string $file_path): array

Description

Retrieves metadata about a file including its size, last modification time, permissions, file name, and MIME type.

Parameters

Parameter Type Description Default
$file_path string The path to the file. N/A

Return Value

Type Description
array Returns an array with file metadata.

Throws

Exception if the file does not exist.

Example Usage

$file_path = '/path/to/your/file.txt';
try {
    $file_info = $this->file->info($file_path);
    print_r($file_info);
} catch (Exception $e) {
    echo 'Error: ' . $e->getMessage();
}
