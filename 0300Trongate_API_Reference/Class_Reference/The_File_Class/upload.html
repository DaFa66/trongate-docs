upload()

public function upload(array $config): array|null

Description

Uploads a file to a specified destination, with optional configurations.

Parameters

Parameter Type Description Default
$config array An array containing upload configurations, including destination, target module, upload to module, and make random name. N/A

Return Value

Type Description
array|null Returns an array containing file information if the upload is successful. Returns null if an error occurs.

Example Usage

$config = [
    'destination' => 'uploads',
    'target_module' => 'your_module',
    'upload_to_module' => true,
    'make_rand_name' => true
];
$file_info = $this->file->upload($config);
if ($file_info !== null) {
    echo "File uploaded successfully: " . $file_info['file_name'];
} else {
    echo "Error uploading file. Please check the configuration and try again.";
}
