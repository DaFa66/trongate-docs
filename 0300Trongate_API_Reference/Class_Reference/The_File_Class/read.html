read()

function read(string $file_path): string

Description

Reads the contents of a file.

Parameters

Parameter Type Description Default
$file_path string The path to the file to be read. N/A

Return Value

Type Description
string Returns the contents of the file.

Throws

Exception if the file does not exist, cannot be read, or access to it is restricted.

Example Usage

$file_path = '/path/to/your/file.txt';
try {
    $content = read($file_path);
    echo $content;
} catch (Exception $e) {
    echo 'Error: ' . $e->getMessage();
}
