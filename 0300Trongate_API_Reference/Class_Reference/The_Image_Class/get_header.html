get_header()

function get_header(): string

Description

This method returns the MIME type corresponding to the image format of the loaded image, which is essential for setting correct HTTP Content-Type headers when serving images directly from PHP scripts. If no image is loaded or the image type has not been set, this method throws a Nothing_loaded_exception to prevent misuse and aid in debugging.

Parameters

This method does not take any parameters.

Exceptions

Throws a Nothing_loaded_exception if no image has been loaded or the image type is not set, indicating that operations requiring an image cannot proceed.

Return Value

Type Description
string Returns the MIME type of the image, suitable for HTTP Content-Type headers, e.g., 'image/jpeg'.

Example Usage

try {
    $mime_type = $image->$this->image->get_header();
    header("Content-Type: " . $mime_type);
} catch (Nothing_loaded_exception $e) {
    echo "Error: " . $e->getMessage();
}
