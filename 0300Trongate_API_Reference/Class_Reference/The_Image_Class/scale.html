scale()

function scale(float $scale): void

Description

Scales the currently loaded image by a specified percentage, adjusting both the width and height while maintaining the image's aspect ratio. The method calculates the new dimensions based on the percentage provided and resizes the image accordingly.

Parameters

Parameter Type Description Default
$scale float The scaling percentage. A value of 100 maintains the original size, values less than 100 decrease the size, and values above 100 increase the size. N/A

Exceptions

Throws an Exception if no image is loaded or if the scale value is non-positive, to ensure safe operation.

Return Value

Type Description
void This method does not return a value but resizes the image in-place.

Example Usage

// Example demonstrating how to scale an image
try {
    // First, load an image
    $this->image->load('path/to/image.jpg');
    // Now scale it by 50%, reducing its size
    $this->image->scale(50);
    echo "Image scaled successfully to 50% of its original size.";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
