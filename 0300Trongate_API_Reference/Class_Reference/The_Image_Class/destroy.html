destroy()

function destroy(): void

Description

This method frees up the memory allocated to the image resource that is currently loaded within the instance. It should be invoked when the image is no longer needed, especially in scripts that process multiple or large images, to prevent memory leaks and manage system resources efficiently. Failure to call this method in such scenarios can lead to increased memory usage and possible performance degradation.

Parameters

This method does not take any parameters.

Return Value

Type Description
void This method does not return a value.

Example Usage

// Assuming $image is an instance of the Image class and an image has been loaded.
$image->destroy();
// After this call, the image resource is freed and the image variable is set to null.
