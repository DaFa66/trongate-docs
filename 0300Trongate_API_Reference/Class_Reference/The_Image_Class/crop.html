crop()

function crop(int $width, int $height, string $trim = 'center'): void

Description

Adjusts the image to the specified width and height by cropping it according to the provided 'trim' parameter. This method allows precise control over the section of the image to retain, either centering or aligning the crop to one side, based on the focus area specified. Ideal for creating uniform image dimensions without distorting the content.

Parameters

Parameter Type Description Default
$width int The desired width of the cropped image. N/A
$height int The desired height of the cropped image. N/A
$trim string Optional. Specifies the focus area of the crop: 'center', 'right', or other values defaulting to no offset. center'

Exceptions

Throws an InvalidArgumentException if the provided dimensions are non-positive or exceed the original image's dimensions, ensuring that parameters are valid for cropping.

Return Value

Type Description
void This method modifies the image in-place and does not return a value.

Example Usage

// Example demonstrating how to crop an image
try {
    // First, load an image
    $this->image->load('path/to/image.jpg');
    // Crop it to 200x200 pixels focusing on the center
    $this->image->crop(200, 200, 'center');
    echo "Image cropped successfully.";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
