get_width()

function get_width(): int

Description

Returns the width of the currently loaded image. This method relies on PHP's GD library function imagesx() to obtain the width of the image resource. An image must be loaded prior to using this method.

Parameters

This method does not require any parameters.

Return Value

Type Description
int The width of the image in pixels, if an image is loaded. Throws an exception if no image is currently loaded.

Example Usage

// Example showing how to load an image and get its width
try {
    // The image is loaded using a filename path; ensure the path is correct
    $this->image->load('path/to/image.jpg');
    $width = $this->image->get_width();
    echo "Width of the image: " . $width . " pixels";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
