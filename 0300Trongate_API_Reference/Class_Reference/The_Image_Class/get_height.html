get_height()

function get_height(): int

Description

Returns the height of the currently loaded image. This method utilizes PHP's GD library function imagesy() to retrieve the height of the image resource. An image must be loaded prior to using this method to ensure valid output.

Parameters

This method does not require any parameters.

Return Value

Type Description
int The height of the image in pixels, if an image is loaded. Throws an exception if no image is currently loaded.

Example Usage

// Example demonstrating how to load an image and retrieve its height
try {
    // The image is loaded using a filename path; ensure the path is correct
    $this->image->load('path/to/image.jpg');
    $height = $this->image->get_height();
    echo "Height of the image: " . $height . " pixels";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
