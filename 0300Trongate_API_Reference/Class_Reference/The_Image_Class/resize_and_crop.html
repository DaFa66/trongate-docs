resize_and_crop()

function resize_and_crop(int $width, int $height): void

Description

Adjusts the image to precisely match specified dimensions by resizing to maintain the aspect ratio and then cropping the excess. This method is ideal for ensuring that the image fits exact dimensions, which is often required for consistent presentation in UI components like thumbnails or profile pictures.

Parameters

Parameter Type Description Default
$width int The desired width of the final image. N/A
$height int The desired height of the final image. N/A

Exceptions

Throws an Exception if no image is loaded, or if the provided width or height are non-positive values, ensuring that operation parameters are valid.

Return Value

Type Description
void This method does not return a value but modifies the image in-place to the specified dimensions.

Example Usage

// Example demonstrating how to resize and crop an image
try {
    // First, load an image
    $this->image->load('path/to/image.jpg');
    // Resize and crop it to 200x200 pixels
    $this->image->resize_and_crop(200, 200);
    echo "Image resized and cropped successfully.";
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
