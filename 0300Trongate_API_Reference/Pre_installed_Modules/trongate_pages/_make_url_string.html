_make_url_str()

function _make_url_str(string $page_title): string

Description

Creates a URL string (slug) from the provided page title. If the generated URL string is not unique, it appends search-friendly strings or a random string until a unique URL string is obtained.

Parameters

Parameter Type Description Default Required
$page_title string The title of the page from which the URL string (slug) will be created. N/A Yes

Return Value

Type Description
string Returns a URL string (slug) based on the provided page title, or a random string if the URL string is not unique.

Summary of _make_url_string_unique()

This method invokes the _make_url_string_unique() method to ensure that the generated URL string is unique. If the initial URL string is found to be non-unique, _make_url_string_unique() appends search-friendly strings or a random string until a unique URL string is obtained.
