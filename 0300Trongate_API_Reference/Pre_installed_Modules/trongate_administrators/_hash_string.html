function _hash_string(string $str): string

Description

Hashes a string using the Bcrypt algorithm.

Parameters

Parameter Type Description Default
$str string The string to be hashed. N/A

Return Value

Type Description
string The hashed string.

Example Usage

echo _hash_string("password123");
// Example output: $2y$11$l7FuqN7Fip5PfZ3LJpGkcu2lFIEfLqCV8eBb5cCYoamP3gKvE6lWu
