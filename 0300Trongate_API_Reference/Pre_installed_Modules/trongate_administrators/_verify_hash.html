function _verify_hash(string $plain_text_str, string $hashed_string): bool

Description

Verifies a plain text string against a hashed string.

Parameters

Parameter Type Description Default
$plain_text_str string The plain text string to verify. N/A
$hashed_string string The hashed string to compare against. N/A

Return Value

Type Description
bool Returns TRUE if the verification is successful, otherwise FALSE.

Example Usage

$plain_text_str = "password123";
$hashed_string = "$2y$11$l7FuqN7Fip5PfZ3LJpGkcu2lFIEfLqCV8eBb5cCYoamP3gKvE6lWu"; // This is a sample hashed string
$result = _verify_hash($plain_text_str, $hashed_string);
// Example output: true
