username_check()

function username_check(string $str): string|bool

Description

Checks the availability of a username and validates it against existing usernames.

Parameters

Parameter Type Description Default
$str string The username to be checked. N/A

Return Value

Type Description
string|bool Returns an error message if the username is not available, otherwise returns TRUE.

Example Usage

$username = "john_doe";
$result = username_check($username);
if ($result !== true) {
    echo $result; // Output: The username that you submitted is not available.
} else {
    echo "Username is available!";
}
