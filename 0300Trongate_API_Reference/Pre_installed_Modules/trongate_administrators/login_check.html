login_check()

function login_check(string $submitted_username): string|bool

Description

Validates the submitted username and password for login authentication.

Parameters

Parameter Type Description Default
$submitted_username string The username submitted for login. N/A

Return Value

Type Description
string|bool Returns an error message (string) if authentication fails, otherwise returns TRUE.

Example Usage

$submitted_username = "john_doe";
$result = login_check($submitted_username);
if ($result !== true) {
    echo $result; // Output: You did not enter a correct username and/or or password.
} else {
    echo "Login successful!";
}
