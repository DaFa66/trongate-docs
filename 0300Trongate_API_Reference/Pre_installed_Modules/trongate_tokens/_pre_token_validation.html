_pre_token_validation()

function _pre_token_validation(array $input): array

Description

Ensures that an input data array contains a valid numeric 'user_id' property before proceeding with token generation. If the environment is not set to 'dev', pre-token validation is disabled, and token issuance is prevented. In such cases, appropriate HTTP response codes are set to indicate the failure reason.

Parameters

Parameter Type Description Default
$input array An array containing input data to be validated. N/A

Return Value

Type Description
array The input data array after validation checks have been performed.
