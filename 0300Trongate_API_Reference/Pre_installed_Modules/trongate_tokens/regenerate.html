regenerate()

function regenerate(): void

Description

Regenerates a token with a new expiration date. In instances of success, the method regenerates a token with a new expiration date and outputs the newly generated token (a 32-character-long string) to the browser. However, if the input format is invalid or the old token does not exist, appropriate HTTP response codes are set to indicate the failure reason.

Parameters

The method does not accept parameters as arguments passed into a function. However, it attempts to read the following parameters from the third and fourth URL segments respectively:

  • $old_token (string): The old token to be regenerated.
  • $expiry_date (int): The new expiration date for the regenerated token.

Return Value

Type Description
void No return value.
