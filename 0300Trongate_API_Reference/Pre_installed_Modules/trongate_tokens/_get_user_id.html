_get_user_id()

function _get_user_id(?string $token = null): int|false

Description

Retrieves the Trongate user ID based on a provided token, session, or cookie. Returns the Trongate user ID (int) if found, or false (bool) if not found.

Parameters

Parameter Type Description Default
$token string|null Optional. The token to retrieve the user ID for. null

Return Value

Type Description
int|false Returns the Trongate user ID if found, or false if not found.
