_fetch_token_obj()

function _fetch_token_obj(string $token): object|false

Description

Fetches a token object from the 'trongate_tokens' table based on a provided token.

A token object typically contains information related to a specific token, such as its ID, the associated user ID, expiry date, and code.

Parameters

Parameter Type Description Default
$token string The token to use for fetching the token object. N/A

Return Value

Type Description
object|false The token object if found, or false if not found.
