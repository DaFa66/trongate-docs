_generate_token()

function _generate_token(array $data): string

Description

Generates a token based on the provided data.

The token is a 32-character random string. If no expiry date is provided, the token's default lifespan is used. By default, token data (which includes; 'id', 'token', 'user_id', expiry_date' and 'code') is stored within the 'trongate_tokens' database table.

Parameters

This method accepts an array of key, value pairs as an argument. The following properties, within the array, are expected:

Property Type Description Default Required
'user_id' int Required. The user's ID. N/A Yes
'expiry_date' int Optional. Unix timestamp for token expiration. Current time + default token lifespan No
'set_cookie' bool Optional. If true, set the token as a cookie. false No
'code' string Optional. Custom code for the token. N/A No

Return Value

Type Description
string The generated token.
