destroy()

function destroy(): void

Description

Destroy a token based on an HTTP POST request.

This method deletes a token from the 'trongate_tokens' table upon receiving a valid HTTP header token.

Parameters

Parameter Type Description Default
No parameters are required for this method.

Return Value

Type Description
void Returns nothing. Outputs message indicating success or failure of token deletion with appropriate HTTP status code.
